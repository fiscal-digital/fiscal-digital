# Warm intros — fundações invitation-only

> Lista de fundações que **não têm edital aberto público** mas financiam projetos cívicos brasileiros via convite ou indicação. Acesso = relacionamento.
>
> **Estratégia:** alavancar a rede do ecossistema OKFN Brasil / Querido Diário (Fiscal Digital se posiciona como extensão, não competidor). Pedir warm intro via comunidade open data Brasil.

## Top 5 priorizado

### 1. Open Knowledge Brasil (OKFN BR) — caminho de entrada

Não é fonte de funding direto, mas é a porta para tudo abaixo. Já é parceiro de fato — Fiscal Digital depende do Querido Diário e linka para todo achado.

- **Ação:** abrir issue de apresentação no repo `okfn-brasil/querido-diario` referenciando uso da API. Pedir reunião com a equipe de Querido Diário (canal Discord OKFN BR ou e-mail equipe@ok.org.br).
- **Pitch:** "Estendemos o ecossistema, não competimos. Como podemos ajudar?"
- **Saídas possíveis:** carta de apoio para grants, intro para Luminate, intro para Knight Center, co-divulgação.

### 2. Luminate Group (ex-Omidyar Network)

Maior funder global de civic tech / open governance. Investidor histórico da Open Knowledge Foundation.

- **Modalidade:** invitation-only, não há edital aberto.
- **Foco no Brasil:** confirmado (já apoiou Nossas e OKFN).
- **Ação:** warm intro via OKFN BR. Não vale frio cold-email.
- **Ticket típico:** US$ 100k–500k para civic tech.
- **Critério-chave:** prova de tração (Fiscal Digital tem 12+ achados reais — base mínima).
- **Link:** https://www.luminategroup.com/investee/open-knowledge

### 3. Ford Foundation — Brazil office

Operações em São Paulo. Foco em civic space, anti-corrupção, justiça social. Em abril/2026 anunciou US$ 60M em grants para "nonpartisan organizations protecting and advancing democracy and the rule of law".

- **Modalidade:** invitation-only, mas mantém Brazil office acessível.
- **Ação:** warm intro via OKFN BR ou via Transparência Brasil (já é grantee Ford).
- **Ticket típico:** US$ 50k–250k multi-ano.
- **Cuidado:** Ford prefere OSCs formalizadas — mover OSC formal para a frente se este caminho avançar.
- **Link:** https://www.fordfoundation.org/our-work-around-the-world/brazil/

### 4. Instituto Ibirapitanga

Fundação brasileira focada em soberania alimentar e equidade racial. Não é match perfeito, mas tem programa de "infraestrutura de incidência" que cobre civic tech.

- **Modalidade:** convite + edital ocasional.
- **Ação:** monitorar editais via [Mapa OSC IPEA](https://mapaosc.ipea.gov.br/editais).
- **Ticket típico:** R$ 50k–200k.
- **Link:** https://ibirapitanga.org.br/

### 5. Knight Foundation

Foca jornalismo + civic tech. Brasil é elegível em alguns programas. Apoiou Civic Tech Field Guide.

- **Modalidade:** mix de editais e convite.
- **Ação:** monitorar página de programas + warm intro via Knight Center for Journalism in the Americas (UT Austin — porta brasileira informal).
- **Ticket típico:** US$ 25k–150k.
- **Link:** https://knightfoundation.org/

## Estratégia de approach

### Fase 1 — Construir capital relacional (T+0 a T+60 dias)

1. Abrir issue pública de apresentação no `okfn-brasil/querido-diario` agradecendo a API e mostrando uso real (links para findings publicados).
2. Entrar no Discord OKFN BR e contribuir antes de pedir.
3. Apresentar projeto em meetup civic tech (Calango Hacker Clube DF, Casa Hacker SP, etc.) — busca passiva de aliados.
4. Postar caso técnico no blog — "Como rodamos 50 cidades em US$ 30/mês com Bedrock".

### Fase 2 — Ativar rede (T+60 a T+120 dias)

1. Pedir warm intro com Luminate via OKFN.
2. Pedir intro com Ford Brazil via Transparência Brasil OU OKFN.
3. Pedir letter of support de OKFN para usar em aplicações.
4. Confirmar fiscal sponsor (Open Source Collective).

### Fase 3 — Converter (T+120 dias em diante)

1. Submeter Mozilla Democracy × AI 2027 (deadline esperado mar/2027).
2. Submeter Fundo Brasil 2027 (deadline esperado mar/2027).
3. Acompanhar GNI Innovation Challenge LatAm próxima abertura.
4. Pitch direto para Luminate / Ford com tração 6+ meses + cartas de apoio.
