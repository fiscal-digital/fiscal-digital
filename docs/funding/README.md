## GitHub Sponsors — passos para Diego (requer web UI)

`.github/FUNDING.yml` já está no repo (mostra botão "Sponsor" nos repos da org). Para completar o cadastro:

2. Clicar em **"Join the waitlist"** (ou "Get started")
3. Preencher dados bancários + tax form (MEI — CNPJ)

> Atenção: o cadastro usa a conta pessoal do Diego como titular, não um e-mail genérico da org.
